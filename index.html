Frontend Development

CSS REACT JAVASCRIPT

Front-end development is the process of building components that interact with users. Examples are the user interface ,buttons, user-entered data, websites, and user experience (UX) feature.


  • HTML is the standard markup language for creating Web pages.

  • CSS is the language we use to style a Web page. What is CSS? CSS stands for Cascading Style Sheets

    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. SASS (Preprocessor)

  • JavaScript is an object-oriented computer programming language commonly used to create interactive effects within web browsers.

  • React is a JavaScript library used for building user interfaces (UI), especially for single-page applications (SPAs). It was developed by Facebook and is widely used in modern web development.
